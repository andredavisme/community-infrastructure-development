About This Dataset

Communities in this dataset are drawn from York County and Cumberland County, Maine, plus state-level aggregates for baseline comparison. Each community record is maintained in the project Supabase database (cid_communities) and linked to available infrastructure data tables.

  • Municipal — incorporated towns or cities with local governance
  • County — county-level aggregate for regional context
  • Regional — state-level baseline (State of Maine)

Water capacity data is currently available for Kennebunkport (KKWD). Additional communities will be added as data is collected and verified. Contributions and corrections welcome via GitHub.
