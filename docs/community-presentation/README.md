# Community Presentation

A four-page GitHub Pages site telling the story of community capacity, infrastructure opportunities, and responsible decision-making in coastal Maine.

**Live site:** https://andredavisme.github.io/community-infrastructure-development/docs/community-presentation/

## Pages

| Page | File | Purpose |
|---|---|---|
| Home | `index.html` | Frames the three core questions every proposal must answer |
| Community Capacity | `capacity.html` | Water, wastewater, environmental, and fiscal capacity with real data |
| Proposal Review | `proposal.html` | Golf course scenario benchmarks vs. community capacity |
| Decision Framework | `framework.html` | Plain-language criteria for responsible infrastructure decisions |

## Data Sources

All figures draw from:
- USGA Water Resource Center
- GCSAA Environmental Stewardship Report 2022
- EPA WaterSense / EJScreen
- Town of York Annual Financial Reports
- Kennebunk Assessor data
- Maine Revenue Services (MRS) MVR Statistical Summary
- KKWD (Kennebunk, Kennebunkport and Wells Water District) capacity estimates

See [`docs/data-dictionary.md`](../data-dictionary.md) and [`docs/glossary.md`](../glossary.md) for full definitions.
