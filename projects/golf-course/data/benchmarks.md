# Golf Course — Benchmark Sources

All values used in seed data files are estimates derived from published benchmarks.
See [`methodology.md`](../../../docs/methodology.md) for full citation approach.

| Metric | Value | Source Basis |
|---|---|---|
| Total acreage | 150 ac | Scenario definition |
| Impervious surface | ~15 ac (~10%) | USGA course design standards |
| Annual water use | ~110M gal/yr | USGA/Golf Course Superintendents avg 70–90 ac-ft/yr for coastal ME climate |
| Peak day water demand | ~312,000 gpd | Annual ÷ 353 irrigating days × 1.0 peak factor |
| Annual electricity | ~1,500 MWh/yr | EPA EnergyStar commercial turf benchmark |
| Permanent FTE jobs | 35 (seasonal) | Industry standard for 18-hole public course |
| Est. annual tax revenue | ~$180,000/yr | Maine mil rate applied to estimated assessed value |
| Est. construction cost | $12,000,000 | RSMeans regional index, coastal Maine |
| Chemical use | Pesticides + fertilizers | Standard golf course IPM program |
| Reversibility | Moderate | Turf to native conversion feasible within 3–5 years |
