# Utility Use Estimates — Golf Course Scenario

All values are estimates. See benchmarks.md for source basis.

## Electricity

| Metric | Value | Notes |
|---|---|---|
| Annual electricity | ~1,500 MWh/yr | Clubhouse, irrigation pumps, lighting, cart charging |
| Peak monthly demand | ~175 MWh (Jul) | Summer operations |
| Off-peak monthly demand | ~60 MWh (Jan) | Minimal heating/maintenance |
| Grid impact | Standard commercial | Existing rural distribution sufficient |
| Estimated annual utility cost | ~$195,000/yr | @ ~$0.13/kWh Maine avg commercial rate |
