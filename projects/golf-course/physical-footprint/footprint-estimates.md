# Physical Footprint — Golf Course Scenario

All values are estimates. See benchmarks.md for source basis.

## Land Use by Category

| Land Use Category | Acreage | % of Total | Notes |
|---|---|---|---|
| Turf (fairways, rough) | ~105 ac | 70% | Managed grass — irrigated |
| Native / naturalized | ~20 ac | 13% | Roughs, out-of-play areas |
| Wetland / buffer | ~5 ac | 3% | 50-ft wetland buffer committed |
| Hardscape (cart paths, parking) | ~8 ac | 5% | Impervious |
| Building (clubhouse, maintenance) | ~2 ac | 1% | Impervious |
| Buffer / setback | ~5 ac | 3% | Road and property line setbacks |
| Stormwater / retention | ~5 ac | 3% | Detention ponds |
| **Total** | **~150 ac** | **100%** | |

## Impervious Surface Summary

| Category | Impervious Acreage |
|---|---|
| Hardscape | ~8 ac |
| Buildings | ~2 ac |
| Compacted turf edges | ~5 ac |
| **Total impervious** | **~15 ac (~10%)** |

## Reversibility

- Rating: **Moderate**
- Turf to native conversion: feasible within 3–5 years with seeding program
- Hardscape removal: standard demolition, no legacy contamination expected
- Clubhouse: permanent structure — low reversibility
- Caveat: if irrigation pond constructed, grading changes are semi-permanent
