# Golf Course Development

Documentation and data tracking for the golf course development initiative in York Harbor, ME.

## Sections

- `water-use/` — Irrigation volumes, source types, conservation measures
- `utility-use/` — Energy, water, waste, and other utility consumption
- `physical-footprint/` — Land use, acreage by type, construction impact
- `environmental-impact/` — Pesticides, runoff, habitat, stormwater
- `employment/` — Jobs created, wage data, local hiring
- `community-engagement/` — Meeting notes, Q&A, public feedback
- `financial-obligations/` — Taxes, fees, municipal contributions, impact bonds
- `milestones/` — Progress log and decision records
