# Golf Course Development — Fictitious Scenario

> ⚠️ **This is a fictitious development scenario.** No actual golf course is being proposed or developed. All analysis uses real data from published industry sources, regulatory benchmarks, and peer-reviewed research. The scenario is structured to be directly transferable to any real coastal or semi-rural golf course development project.

## Scenario Description

A proposed 18-hole golf course on approximately 150 acres in a coastal Maine setting. The scenario models a mid-range public/semi-private course with a clubhouse, maintenance facility, and parking.

## Analysis Sections

- `water-use/` — Irrigation volumes, source types, conservation measures
- `utility-use/` — Energy, water, waste, and other utility consumption
- `physical-footprint/` — Land use, acreage by category, construction impact
- `environmental-impact/` — Pesticides, runoff, habitat, stormwater
- `employment/` — Jobs created, wage data, local hiring
- `community-engagement/` — Modeled public meeting scenarios, Q&A, feedback
- `financial-obligations/` — Taxes, fees, municipal contributions, impact bonds
- `community-context/` — Population, nearby communities, municipal finances, trends
- `milestones/` — Analysis progress log

## Analysis Perspectives

All impact dimensions are analyzed at multiple scales:

| Scale | Description |
|-------|-------------|
| **Individual** | Per-capita impact on residents (cost, benefit, exposure) |
| **Aggregate — Host Community** | Total impact on the primary host municipality |
| **Aggregate — Regional** | Impact across nearby communities within the service/impact radius |
| **Fiscal** | Effect on municipal tax income and budgeted expenses |
| **Trend** | 25-year historical context where data is available (2000–2025) |
