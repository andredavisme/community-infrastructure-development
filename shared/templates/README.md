# Reusable Templates

Standardized templates for data entry, reporting, and community presentation across all projects.

## Available Templates

- `utility-log-template.md` — Logging utility consumption events
- `footprint-survey-template.md` — Recording physical land use
- `employment-report-template.md` — Tracking job creation and wages
- `community-meeting-template.md` — Documenting public engagement sessions
- `financial-obligations-template.md` — Recording fees, taxes, and contributions
